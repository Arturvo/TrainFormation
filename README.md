# TrainFormation
This repository contains source code of a Unity project TrainFormation. TrainFormation is a simple, fast-paced puzzle game created by one student as a university project. Place rails in front of an ever-accelerating train to cross procedurally generated levels, reach stations, collect power ups and make it as far as you can!

TrainFormation is available to play for free on steam:
https://store.steampowered.com/app/2077150/TrainFormation/

The build can also be downloaded directly: 
https://arturvo.itch.io/trainformation
